## What is Geocoder ID?

Geocoder ID converts geographic coordinates (latitude/longitude) into human-readable administrative addresses for locations in Indonesia. It provides detailed location data from province level down to village/sub-district (kelurahan/desa) level - all without requiring an internet connection after initial setup.

For example, coordinates `-6.2088, 106.8456` become:
- **Province**: DKI Jakarta
- **Regency**: Kota Administrasi Jakarta Selatan
- **District**: Setiabudi
- **Village**: Pasar Manggis

## How It Works

Geocoder ID uses a DuckDB spatial database with pre-loaded Indonesian administrative boundary data. When you query a coordinate:

1. The spatial index quickly identifies which polygon contains your point
2. The hierarchical lookup retrieves all administrative levels (province → regency → district → village)
3. Results are returned with both official codes and names

All queries run locally on your machine with no external API calls.

## Administrative Levels

Geocoder ID provides data for all four administrative levels in Indonesia:

| Level | Indonesian Term | Example |
|-------|----------------|---------|
| Province | Provinsi | DKI Jakarta |
| Regency/City | Kabupaten/Kota | Kota Administrasi Jakarta Selatan |
| District | Kecamatan | Setiabudi |
| Village/Sub-district | Kelurahan/Desa | Pasar Manggis |

Each level includes both the official administrative code and name.
